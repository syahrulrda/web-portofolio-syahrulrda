Web Programming

Pembelajaran dasar-dasar pemrograman web.

Deskripsi

Pelatihan Web Programming ini berfokus pada pengenalan konsep dasar pembuatan website mulai dari HTML, CSS, hingga JavaScript. Materi ini membantu peserta memahami struktur halaman web, cara membuat tampilan yang responsif, serta dasar interaksi pengguna. Pelatihan ini sangat cocok bagi pemula yang ingin memulai perjalanan di dunia pengembangan web.

Fitur Pembelajaran:
  • Pengenalan HTML (struktur halaman web)
  • Pengenalan CSS (membuat tampilan lebih menarik)
  • Dasar JavaScript untuk interaktivitas
  • Latihan membuat halaman web sederhana
  • Pemahaman dasar responsive design
  • Pengenalan tools pengembangan web
